## Descripción
Es una aplicación hecha con Spring Boot para gestionar información sobre conflictos bélicos.

## Instrucciones
Tener la version de Java 17, clonar el repositorio del github y abrirlo en un IDE como por ejemplo Intellij.
Para ejecutar la app hay que ir a ConflictTrackerApp y darle a Run.
